Portugol Studio

O Portugol Studio é uma IDE e uma linguagem de programação totalmente gratuita, focada principalmente na introdução à pensamento computacional e programação. Com ela é possível criar jogos, interfaces e até realizar integrações com serviços REST de maneira simples.

Atualmente não faço mais parte da equipe do projeto, mas fui o criador das bibliotecas para criação e interação com Objetos JSON e conexão com serviços REST, além de ter palestrado utilizando o Azure Cognitive Services via Portugol no TDC Florianópolis.

Você pode fazer o download da ferramenta gratuitamente neste link.
